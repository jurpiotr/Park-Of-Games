# Park-Of-Games
Application with API RAWG.IO

* You need install npm

# Getting started

1. Clone the repository
```sh
git clone https://github.com/jurpiotr/Park-Of-Games.git
```
```sh
cd Park-Of-Games
```

2. Install:
```sh
npm install
```

3. Run Server:
```sh
 npm run watch
```

Visit http://localhost:8080 to see application.
